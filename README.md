### `jun_generator` 代码生成器

### 项目说明
jun_generato，基于SpringBoot+Freemarker的代码生成器，可以生产单表CRUD完整的操作

### 功能清单 基础篇(工具)：代码生成器（jun_code_generator）

> SpringBoot系开发框架组件，基于SpringBoot微服务开发组件，新企业级REST服务

【jun_code_generator】[Java代码生成模块,数据库表代码生成、页面代码生成](https://github.com/wujun728/jun_code_generator/jun_code_helper)
【jun_code_mybatis】[Mybatis代码生成模块](https://github.com/wujun728/jun_code_generator)
【jun_code_mybatisplus】[MybatisPlus代码生成模块](https://github.com/wujun728/jun_code_generator)
